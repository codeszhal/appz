# Currency Total

A lightweight static PWA for local multi-currency records.

- Separate collapsible tables for IDR, CNY, and USDT
- Per-currency totals with no conversion
- LocalStorage persistence
- Export/import JSON backups
- Offline-capable service worker and web app manifest

Open `index.html` directly for a quick preview, or serve the folder over HTTP to test the PWA/service worker behavior.
